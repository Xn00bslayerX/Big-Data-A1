# Big-Data-A1
To run the file, first install the required libraries using the command:
```pip install -r requirements.txt
```
You should have Python 3.8 or higher installed on your system.
Then, you can run the Jupyter notebook `assignment1.ipynb` to see the data analysis and visualizations. This will download the necessary data files, perform the analysis, and generate the visualizations as part of the assignment.
